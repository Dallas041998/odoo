# Todo List Module

This module provides a comprehensive todo list application for Odoo with both server-side and client-side functionality.

## Features

* Server-side task management with full Odoo integration
* Client-side interface with localStorage for offline functionality  
* Synchronization between client and server
* User-specific task security
* Modern responsive interface

## Installation

Install like any other Odoo module. The module depends on 'base' and 'web' modules which are included in standard Odoo installations.

## Usage

After installation, access the module via Apps → Todo List or through the main menu.
